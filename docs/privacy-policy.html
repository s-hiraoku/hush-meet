プライバシーポリシー

最終更新日: 2026年3月21日

概要

Hush Meet（以下「本拡張機能」）は、Google Meet での通話中に発話を検出し、自動的にミュートの切り替えを行う Chrome 拡張機能です。本拡張機能はユーザーのプライバシーを最大限尊重し、最小限のデータのみを使用します。

収集するデータ

本拡張機能はユーザーの個人データを一切収集・送信しません。

マイクへのアクセス

本拡張機能は発話検出のためにマイクへのアクセスを必要とします。マイクから取得した音声データについて：

  • 音声データは端末内でのみリアルタイムに処理されます
  • 音声データは録音・保存されません
  • 音声データは外部サーバーに送信されません
  • 音声の音量レベル（RMS値）のみを計算に使用し、音声の内容は一切解析しません

ローカルストレージ

本拡張機能は Chrome Storage API を使用して、以下の設定情報をローカルに保存します：

  • 動作モード（Off / Auto / Auto-Off / Push-to-Talk）
  • 発話検出の感度設定
  • 猶予時間の設定
  • ショートカットキーの設定
  • 選択中のマイクデバイス
  • テーマ・言語の設定

これらのデータはお使いのブラウザ内にのみ保存され、外部に送信されることはありません。

アクセスするウェブサイト

本拡張機能は https://meet.google.com でのみ動作します。その他のウェブサイトではコンテンツスクリプトは実行されません。

第三者への情報提供

本拡張機能は第三者にいかなるデータも提供しません。分析ツール、トラッキングコード、広告は一切含まれていません。

オープンソース

本拡張機能のソースコードは GitHub で公開されています。データの取り扱いについてご不明な点がある場合は、ソースコードをご確認いただけます。

お問い合わせ

変更について

本プライバシーポリシーは、必要に応じて更新される場合があります。重要な変更がある場合は、本ページにて通知します。
