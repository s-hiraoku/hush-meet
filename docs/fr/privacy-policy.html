Politique de Confidentialité

Dernière mise à jour : 21 mars 2026

Aperçu

Hush Meet (ci-après « cette extension ») est une extension Chrome qui détecte la parole pendant les appels Google Meet et active/désactive automatiquement le micro. Cette extension respecte la vie privée des utilisateurs dans toute la mesure du possible et n'utilise que le minimum de données nécessaire.

Collecte de données

Cette extension ne collecte ni ne transmet aucune donnée personnelle.

Accès au microphone

Cette extension nécessite l'accès au microphone pour la détection vocale. Concernant les données audio obtenues à partir du microphone :

  • Les données audio sont traitées en temps réel uniquement sur votre appareil
  • Les données audio ne sont jamais enregistrées ni stockées
  • Les données audio ne sont jamais envoyées à des serveurs externes
  • Seul le niveau de volume audio (valeur RMS) est utilisé pour le calcul ; le contenu audio n'est jamais analysé

Stockage local

Cette extension utilise l'API Chrome Storage pour stocker localement les paramètres suivants :

  • Mode de fonctionnement (Désactivé / Auto / Auto-Off / Push-to-Talk)
  • Paramètres de sensibilité de détection vocale (par mode)
  • Paramètres de délai de grâce (par mode)
  • Paramètres des raccourcis clavier
  • Périphérique de microphone sélectionné
  • Paramètres de thème et de langue

Ces données sont stockées uniquement dans votre navigateur et ne sont jamais transmises à l'extérieur.

Accès aux sites web

Cette extension fonctionne uniquement sur https://meet.google.com. Les scripts de contenu ne sont exécutés sur aucun autre site web.

Divulgation à des tiers

Cette extension ne fournit aucune donnée à des tiers. Elle ne contient aucun outil d'analyse, code de suivi ou publicité.

Open Source

Le code source de cette extension est disponible sur GitHub. Si vous avez des questions sur la gestion des données, vous pouvez consulter directement le code source.

Contact

Si vous avez des questions concernant cette politique de confidentialité, veuillez nous contacter via GitHub Issues.

Modifications

Cette politique de confidentialité peut être mise à jour si nécessaire. En cas de modifications importantes, nous vous en informerons sur cette page.
