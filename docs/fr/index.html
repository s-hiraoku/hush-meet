Hush Meet

Une extension Chrome qui coupe automatiquement votre microphone sur Google Meet lorsque vous ne parlez pas, empêchant les bruits de fond d'atteindre les autres participants.

Fonctionnalités

  • Réactivation automatique lorsque la parole est détectée, mise en sourdine automatique après que vous cessez de parler
  • 4 modes de fonctionnement :
    • Désactivé — Extension désactivée, comportement par défaut de Meet
    • Auto — Réactivation déclenchée par la parole et mise en sourdine automatique (pour les environnements calmes)
    • Auto-Off — Réactivation uniquement par le contrôle manuel de Meet ou le raccourci, mise en sourdine automatique après la parole (pour les environnements bruyants)
    • Appuyer pour parler — Maintenez la touche de raccourci enfoncée pour parler
  • Touche de raccourci personnalisable (par défaut : Ctrl+Shift+M) — disponible dans tous les modes sauf Désactivé
  • Surveillance audio indépendante (flux micro séparé de Google Meet)
  • Porte de bruit pondérée par la voix — met en valeur les fréquences de la voix humaine (300 Hz–3 kHz), réduisant les déclenchements intempestifs causés par le clavier, le ventilateur et le bruit ambiant
  • Conception de seuil asymétrique pour éviter les oscillations
  • Indicateur de niveau audio en temps réel et analyseur de spectre
  • Sensibilité et délai de grâce réglables
  • Sécurité de mise en sourdine manuelle — si vous cliquez sur le bouton de mise en sourdine de Meet, l'extension passe automatiquement en mode Désactivé pour éviter les interférences
  • Sélecteur de microphone — choisissez le micro à utiliser
  • 4 thèmes visuels (Par défaut / Radio Analogique / Boombox / Rétro Futur)
  • Icône de barre d'outils adaptée au mode — la couleur et le badge de l'icône changent en fonction du mode actif
  • Prise en charge de l'anglais et du japonais avec changement manuel
  • Vous vous ennuyez en réunion ? Des mini-jeux rétro-futuristes sont inclus (Breakout / Space Invaders / 2048 / Flappy Bird). Attention : les effets sonores des jeux peuvent déclencher la réactivation automatique. Réglez bien votre sensibilité et profitez du frisson de ne pas vous faire prendre.

Comment ça fonctionne

[Entrée Micro] → [Web Audio API (RMS)] → [Détection de la parole] → [Contrôle du bouton Sourdine de Meet]

Un flux micro séparé surveille les niveaux audio. Lorsque la parole est détectée, l'extension clique automatiquement sur le bouton de sourdine de Meet pour réactiver le micro. Lorsque le silence revient, elle remet en sourdine après un délai de grâce configurable.

Sécurité de Mise en Sourdine Manuelle

Lorsque vous mettez en sourdine ou réactivez manuellement via le bouton Meet, l'extension passe automatiquement en mode Désactivé. Cela empêche l'extension de réactiver votre microphone de manière inattendue après l'avoir volontairement mis en sourdine.

  • Surveille le bouton de sourdine de Meet via MutationObserver
  • Distingue les changements de sourdine initiés par l'extension de ceux initiés par l'utilisateur
  • Clic sur le bouton de sourdine manuel → l'extension passe en mode Désactivé
  • Utilisez le raccourci clavier (Ctrl+Shift+M) pour contrôler la sourdine tout en gardant l'extension active

Premiers pas

  1. Installer depuis le Chrome Web Store
  2. Rejoindre un appel Google Meet
  3. Cliquer sur l'icône Hush Meet dans la barre d'outils
  4. Choisir n'importe quel mode sauf Désactivé
  5. Autoriser l'accès au microphone lorsque demandé
  6. Utilisez le raccourci (Ctrl+Shift+M) pour la sourdine — cliquer sur le bouton sourdine de Meet passera l'extension en mode Désactivé

Stack technique

  • Chrome Extension Manifest V3
  • React 19 + TypeScript
  • Vite + CRXJS Vite Plugin
  • Web Audio API (AnalyserNode)
  • Chrome Storage API
