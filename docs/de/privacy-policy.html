Datenschutzrichtlinie

Zuletzt aktualisiert: 21. März 2026

Überblick

Hush Meet (im Folgenden „diese Erweiterung“) ist eine Chrome-Erweiterung, die während Google Meet-Anrufen Sprache erkennt und automatisch die Stummschaltung umschaltet. Diese Erweiterung respektiert die Privatsphäre der Nutzer in größtmöglichem Umfang und verwendet nur die minimal erforderliche Datenmenge.

Datenerfassung

Diese Erweiterung erfasst oder überträgt keine personenbezogenen Daten.

Mikrofonzugriff

Diese Erweiterung benötigt Mikrofonzugriff zur Spracherkennung. Bezüglich der vom Mikrofon erhaltenen Audiodaten:

  • Audiodaten werden nur in Echtzeit auf Ihrem Gerät verarbeitet
  • Audiodaten werden niemals aufgezeichnet oder gespeichert
  • Audiodaten werden niemals an externe Server gesendet
  • Nur der Audiolautstärkepegel (RMS-Wert) wird zur Berechnung verwendet; der Audioinhalt wird nie analysiert

Lokaler Speicher

Diese Erweiterung verwendet die Chrome Storage API, um folgende Einstellungen lokal zu speichern:

  • Betriebsmodus (Aus / Auto / Auto-Aus / Push-to-Talk)
  • Einstellungen der Spracherkennungsempfindlichkeit (pro Modus)
  • Nachklingzeit-Einstellungen (pro Modus)
  • Tastenkürzel-Einstellungen
  • Ausgewähltes Mikrofongerät
  • Theme- und Spracheinstellungen

Diese Daten werden nur in Ihrem Browser gespeichert und niemals extern übertragen.

Website-Zugriff

Diese Erweiterung funktioniert nur auf https://meet.google.com. Content Scripts werden auf keiner anderen Website ausgeführt.

Weitergabe an Dritte

Diese Erweiterung gibt keine Daten an Dritte weiter. Sie enthält keine Analysetools, Tracking-Code oder Werbung.

Open Source

Der Quellcode dieser Erweiterung ist auf GitHub verfügbar. Bei Fragen zum Umgang mit Daten können Sie den Quellcode direkt einsehen.

Kontakt

Bei Fragen zu dieser Datenschutzrichtlinie kontaktieren Sie uns bitte über GitHub Issues.

Änderungen

Diese Datenschutzrichtlinie kann bei Bedarf aktualisiert werden. Bei wesentlichen Änderungen werden wir Sie auf dieser Seite benachrichtigen.
