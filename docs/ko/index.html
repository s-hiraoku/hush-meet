Hush Meet

Google Meet에서 발언하지 않을 때 자동으로 마이크를 음소거하여 배경 소음이 다른 참가자에게 전달되는 것을 방지하는 Chrome 확장 프로그램입니다.

기능

  • 음성이 감지되면 자동으로 음소거 해제, 발언이 끝나면 자동으로 음소거
  • 4가지 작동 모드:
    • Off — 확장 프로그램 비활성화, Meet 기본 동작
    • Auto — 음성 감지 시 자동 음소거 해제 및 자동 음소거 (조용한 환경용)
    • Auto-Off — Meet 수동 조작 또는 단축키로만 음소거 해제, 발언 후 자동 음소거 (시끄러운 환경용)
    • Push-to-Talk — 단축키를 누르고 있는 동안만 발언 가능
  • 사용자 지정 단축키 (기본값: Ctrl+Shift+M) — Off 모드를 제외한 모든 모드에서 사용 가능
  • 독립적인 오디오 모니터링 (Google Meet과 별도의 마이크 스트림)
  • 음성 가중 노이즈 게이트 — 사람의 음성 주파수(300Hz–3kHz)를 강조하여 키보드, 팬, 주변 소음으로 인한 오작동을 줄임
  • 비대칭 임계값 설계로 채터링 방지
  • 실시간 오디오 레벨 미터 및 스펙트럼 분석기
  • 감도 및 유예 기간 조절 가능
  • 수동 음소거 감지 — 선택한 모드를 변경하지 않고 Meet 측 음소거/음소거 해제를 추적
  • 마이크 장치 선택기 — 사용할 마이크 선택 가능
  • 4가지 비주얼 테마 (Default / Analog Radio / Boombox / Retro Future)
  • 모드 인식 툴바 아이콘 — 활성 모드에 따라 아이콘 색상과 배지가 변경
  • 영어 및 일본어 지원, 수동 전환 가능
  • 회의가 지루하신가요? 레트로 퓨처 미니 게임이 포함되어 있습니다 (Breakout / Space Invaders / 2048 / Flappy Bird). 주의: 게임 효과음이 자동 음소거 해제를 트리거할 수 있습니다. 감도를 적절히 설정하고 들키지 않는 스릴을 즐겨보세요.

작동 원리

[마이크 입력] → [Web Audio API (RMS)] → [음성 감지] → [Meet 음소거 버튼 제어]

별도의 마이크 스트림이 오디오 레벨을 모니터링합니다. 음성이 감지되면 확장 프로그램이 자동으로 Meet 음소거 버튼을 클릭하여 음소거를 해제합니다. 무음 상태로 돌아오면 설정된 유예 기간 후 다시 음소거합니다.

수동 음소거 감지

Google Meet에서 수동으로 음소거하거나 음소거를 해제하면, 확장 프로그램은 선택된 모드를 유지하고 내부 발언 상태만 업데이트합니다.

  • MutationObserver를 통해 Meet 음소거 버튼을 모니터링
  • 확장 프로그램에 의한 음소거 변경과 사용자에 의한 음소거 변경을 구분
  • 수동 음소거는 `MUTED`, 수동 음소거 해제는 `SPEAKING`으로 추적
  • 선택된 모드를 변경하지 않음
