개인정보 처리방침

최종 업데이트: 2026년 3월 21일

개요

Hush Meet(이하 "본 확장 프로그램")은 Google Meet 통화 중 음성을 감지하여 자동으로 음소거를 전환하는 Chrome 확장 프로그램입니다. 본 확장 프로그램은 사용자의 개인정보를 최대한 존중하며, 필요한 최소한의 데이터만 사용합니다.

데이터 수집

본 확장 프로그램은 어떠한 개인 데이터도 수집하거나 전송하지 않습니다.

마이크 접근

본 확장 프로그램은 음성 감지를 위해 마이크 접근 권한이 필요합니다. 마이크에서 얻은 오디오 데이터에 대해:

  • 오디오 데이터는 사용자의 기기에서만 실시간으로 처리됩니다
  • 오디오 데이터는 녹음되거나 저장되지 않습니다
  • 오디오 데이터는 외부 서버로 전송되지 않습니다
  • 오디오의 음량 수준(RMS 값)만 연산에 사용되며, 오디오 내용은 분석되지 않습니다

로컬 저장소

본 확장 프로그램은 Chrome Storage API를 사용하여 다음 설정을 로컬에 저장합니다:

  • 작동 모드 (Off / Auto / Auto-Off / Push-to-Talk)
  • 음성 감지 민감도 설정 (모드별)
  • 유예 시간 설정 (모드별)
  • 단축키 설정
  • 선택된 마이크 장치
  • 테마 및 언어 설정

이 데이터는 브라우저 내에만 저장되며, 외부로 전송되지 않습니다.

웹사이트 접근

본 확장 프로그램은 https://meet.google.com에서만 작동합니다. 다른 웹사이트에서는 콘텐츠 스크립트가 실행되지 않습니다.

제3자 정보 제공

본 확장 프로그램은 제3자에게 어떠한 데이터도 제공하지 않습니다. 분석 도구, 추적 코드 또는 광고를 포함하지 않습니다.

오픈 소스

본 확장 프로그램의 소스 코드는 GitHub에서 확인할 수 있습니다. 데이터 처리 방식에 대해 궁금한 점이 있으시면 소스 코드를 직접 검토하실 수 있습니다.

문의

본 개인정보 처리방침에 대해 궁금한 점이 있으시면 GitHub Issues를 통해 문의해 주시기 바랍니다.

변경 사항
