Hush Meet

A Chrome extension that automatically mutes your microphone on Google Meet when you're not speaking, preventing background noise from reaching other participants.

Features

  • Auto-unmutes when speech is detected, auto-mutes after you stop speaking
  • 4 operating modes:
    • Off — Extension disabled, Meet default behavior
    • Auto — Speech-triggered unmute & auto-mute (for quiet environments)
    • Auto-Off — Unmute only by Meet manual control or shortcut, auto-mute after speech (for noisy environments)
    • Push-to-Talk — Hold shortcut key to speak
  • Customizable shortcut key (default: Ctrl+Shift+M) — available in all modes except Off
  • Independent audio monitoring (separate mic stream from Google Meet)
  • Voice-weighted noise gate — emphasizes human voice frequencies (300Hz–3kHz), reducing false triggers from keyboard, fan, and ambient noise
  • Asymmetric threshold design prevents chattering
  • Real-time audio level meter & spectrum analyzer
  • Adjustable sensitivity and grace period
  • Manual mute safety — if you click Meet's mute button directly, the extension automatically switches to Off mode to avoid interference
  • Microphone device selector — choose which mic to use
  • 4 visual themes (Default / Analog Radio / Boombox / Retro Future)
  • Mode-aware toolbar icon — icon color and badge change based on active mode
  • English & Japanese language support with manual switching
  • Bored in a meeting? Retro-future mini-games included (Breakout / Space Invaders / 2048 / Flappy Bird). Fair warning: game sound effects can trigger auto-unmute. Set your sensitivity just right and enjoy the thrill of not getting caught.

How It Works

[Mic Input] → [Web Audio API (RMS)] → [Speech Detection] → [Meet Mute Button Control]

A separate microphone stream monitors audio levels. When speech is detected, the extension automatically clicks the Meet mute button to unmute. When silence returns, it re-mutes after a configurable grace period.

Manual Mute Safety

When you manually mute or unmute via the Meet mute button, the extension automatically switches to Off mode. This prevents the extension from unexpectedly turning your mic back on after you intentionally muted it.

  • Monitors the Meet mute button via MutationObserver
  • Distinguishes extension-initiated vs. user-initiated mute changes
  • Manual mute button click → extension switches to Off mode
  • Use the shortcut key (Ctrl+Shift+M) to control mute while keeping the extension active

Getting Started

  1. Install from the Chrome Web Store
  2. Join a Google Meet call
  3. Click the Hush Meet icon in the toolbar
  4. Choose any mode except Off
  5. Allow microphone access when prompted
  6. Use the shortcut key (Ctrl+Shift+M) to toggle mute — clicking Meet's mute button directly will switch the extension to Off mode
