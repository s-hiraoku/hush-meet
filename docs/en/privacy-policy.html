Privacy Policy

Last updated: March 21, 2026

Overview

Hush Meet (hereinafter "this extension") is a Chrome extension that detects speech during Google Meet calls and automatically toggles mute. This extension respects user privacy to the fullest extent and uses only the minimum amount of data necessary.

Data Collection

This extension does not collect or transmit any personal data.

Microphone Access

This extension requires microphone access for speech detection. Regarding the audio data obtained from the microphone:

  • Audio data is processed in real time only on your device
  • Audio data is never recorded or stored
  • Audio data is never sent to external servers
  • Only the audio volume level (RMS value) is used for computation; the content of the audio is never analyzed

Local Storage

This extension uses the Chrome Storage API to store the following settings locally:

  • Operating mode (Off / Auto / Auto-Off / Push-to-Talk)
  • Speech detection sensitivity settings
  • Grace period settings
  • Shortcut key settings
  • Selected microphone device
  • Theme and language settings

This data is stored only within your browser and is never transmitted externally.

Website Access

This extension operates only on https://meet.google.com. Content scripts are not executed on any other websites.

Third-Party Disclosure

This extension does not provide any data to third parties. It contains no analytics tools, tracking code, or advertisements.

Open Source

The source code of this extension is available on GitHub. If you have any questions about how data is handled, you can review the source code directly.

Contact

If you have questions about this privacy policy, please contact us via GitHub Issues.

Changes

This privacy policy may be updated as needed. If there are significant changes, we will notify you on this page.
