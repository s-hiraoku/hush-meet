隐私政策

最后更新：2026年3月21日

概述

Hush Meet（以下简称"本扩展"）是一款 Chrome 扩展程序，可在 Google Meet 通话期间检测语音并自动切换静音。本扩展最大限度地尊重用户隐私，仅使用最少的必要数据。

数据收集

本扩展不会收集或传输任何个人数据。

麦克风访问

本扩展需要麦克风权限用于语音检测。关于从麦克风获取的音频数据：

  • 音频数据仅在您的设备上进行实时处理
  • 音频数据绝不会被录制或存储
  • 音频数据绝不会发送到外部服务器
  • 仅使用音频音量级别（RMS 值）进行计算，绝不会分析音频内容

本地存储

本扩展使用 Chrome Storage API 在本地存储以下设置：

  • 运行模式（关闭 / 自动 / 自动关闭 / 按键通话）
  • 语音检测灵敏度设置
  • 宽限期设置
  • 快捷键设置
  • 已选择的麦克风设备
  • 主题和语言设置

这些数据仅存储在您的浏览器中，绝不会传输到外部。

网站访问

本扩展仅在 https://meet.google.com 上运行。内容脚本不会在任何其他网站上执行。

第三方披露

本扩展不会向第三方提供任何数据。它不包含任何分析工具、跟踪代码或广告。

开源

本扩展的源代码可在 GitHub 上获取。如果您对数据处理方式有任何疑问，可以直接查看源代码。

联系方式

如果您对本隐私政策有任何疑问，请通过 GitHub Issues 与我们联系。

变更

本隐私政策可能会根据需要进行更新。如有重大变更，我们将在此页面上通知您。
