Política de Privacidad

Última actualización: 21 de marzo de 2026

Descripción general

Hush Meet (en adelante "esta extensión") es una extensión de Chrome que detecta la voz durante las llamadas de Google Meet y activa o desactiva el silencio automáticamente. Esta extensión respeta la privacidad del usuario en la mayor medida posible y utiliza solo la cantidad mínima de datos necesaria.

Recopilación de datos

Esta extensión no recopila ni transmite ningún dato personal.

Acceso al micrófono

Esta extensión requiere acceso al micrófono para la detección de voz. Con respecto a los datos de audio obtenidos del micrófono:

  • Los datos de audio se procesan en tiempo real únicamente en su dispositivo
  • Los datos de audio nunca se graban ni se almacenan
  • Los datos de audio nunca se envían a servidores externos
  • Solo se utiliza el nivel de volumen del audio (valor RMS) para el cálculo; el contenido del audio nunca se analiza

Almacenamiento local

Esta extensión utiliza la API de Chrome Storage para almacenar las siguientes configuraciones de forma local:

  • Modo de operación (Desactivado / Automático / Auto-Desactivado / Pulsar para hablar)
  • Configuración de sensibilidad de detección de voz (por modo)
  • Configuración del período de gracia (por modo)
  • Configuración de teclas de acceso rápido
  • Dispositivo de micrófono seleccionado
  • Configuración de tema e idioma

Estos datos se almacenan únicamente en su navegador y nunca se transmiten externamente.

Acceso a sitios web

Esta extensión opera únicamente en https://meet.google.com. Los scripts de contenido no se ejecutan en ningún otro sitio web.

Divulgación a terceros

Esta extensión no proporciona ningún dato a terceros. No contiene herramientas de análisis, código de seguimiento ni publicidad.

Código abierto

El código fuente de esta extensión está disponible en GitHub. Si tiene alguna pregunta sobre cómo se manejan los datos, puede revisar el código fuente directamente.

Contacto

Si tiene preguntas sobre esta política de privacidad, contáctenos a través de GitHub Issues.

Cambios

Esta política de privacidad puede actualizarse según sea necesario. Si hay cambios significativos, se lo notificaremos en esta página.
